2021年04月02日 371 字 大概 1 分钟

git常用操作

创建仓库命令

命令 解释
git init 初始化仓库
git clone 拷贝一份远程仓库，也就是下载一个项目。

提交与修改

命令 解释
git add 添加文件到仓库
git status 查看仓库当前的状态，显示有变更的文件。
git diff 比较文件的不同，即暂存区和工作区的差异。
git commit 提交暂存区到本地仓库。
git reset 回退版本。
git rm 删除工作区文件。
git mv 移动或重命名工作区文件。

提交日志

命令 解释
git log 查看历史提交记录, 用 –oneline 选项来查看历史记录的简洁的版本。用 –reverse 参数来逆向显示所有日志。
git blame <fileName> 以列表形式查看指定文件的历史修改记录

远程操作

命令 解释
git remote 远程仓库操作
git fetch 从远程获取代码库
git pull 下载远程代码并合并
git push 上传远程代码并合并

分支管理

命令 解释
git branch (branchname) 创建分支, 添加-b参数可以在创建分支之后自动切换, 添加-d参数意为删除分支
git checkout (branchname) 切换本地分支
git merge (branchname) 合并branchname到当前分支, 当出现文件冲突时, 可以使用git add来告诉git冲突已经被解决
git rebase (branchname) 这里不做详解, 详情点击查看
git status 查看当前文件的状态
