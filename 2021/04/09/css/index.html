css

2021年04月09日 425 字 大概 2 分钟

对css做一些简单的小记录

有关于css, 其实没什么好说的, 熟练度的问题, 用的多了, 自然就记住了

本人实际项目中运用scss 多余 less, 所以scss相对熟练, 但是两者没什么大的差别, 关键字略微不同

不管是引用了less和scss, 都要对css全局样式进行封装, 避免重复的样式重复写多遍

css布局中, flex布局使用的相对熟练, flex布局中, 主轴会在水平和垂直之间变动, justify-content属性也会随着主轴变为控制水平和垂直的样式

再然后就是记录一些全局样式

/*垂直居中*/
.vertical-middle {
  display: -webkit-flex;
  display: flex;
  -webkit-align-items: center;
  align-items: center;
}

/*垂直水平居中*/
.horizontal-vertical-middle{
  display: -webkit-flex;
  display: flex;
  -webkit-align-items: center;
  align-items: center;
  -webkit-justify-content: center;
  justify-content: center;
}

/*垂直居中，水平靠右*/
.vertical-middle-horizontal-right {
  display: -webkit-flex;
  display: flex;
  -webkit-align-items: center;
  align-items: center;
  -webkit-justify-content: flex-end;
  justify-content: flex-end;
}
/*水平居中，垂直靠下*/
.horizontal-middle-vertical-bottom{
  display: -webkit-flex;
  display: flex;
  -webkit-align-items: flex-end;
  align-items: flex-end;
  -webkit-justify-content: center;
  justify-content: center;
}


.flex{
  display: -webkit-flex;
  display: flex;
}

//水平布局
.horizontal-layout{
  flex-direction: row;
  .flex;
}

//垂直布局
.vertical-layout{
  flex-direction: column;
  .flex;
}

//水平居中
.horizontal-middle{
  justify-content: center;
  display: flex;
}


.x-axis-horizontal-middle{
  justify-content: center;
}

.x-axis-vertical-middle{
  align-items: center;
}

//y轴方向 水平居中
.y-axis-horizontal-middle{
  align-items: center;
}

//y轴方向 垂直居中
.y-axis-vertical-middle{
  justify-content: center;
}

//换行
.flex-wrap{
  flex-wrap:wrap;
}

//兼容h5 Image是由div包裹img组成
.Image{
  .vertical-layout;
  .y-axis-vertical-middle;
  img {
    width: 100%;
  }
}
